# Folder Images

Folder ini digunakan untuk menyimpan gambar-gambar website SMK Metland Cileungsi.

## Gambar yang Diperlukan

### Logo & Branding
- `logo-smk-metland.png` - Logo sekolah (disarankan: 200x60px)
- `favicon.png` - Favicon website (disarankan: 32x32px atau 64x64px)
- `og-image.jpg` - Open Graph image untuk social media sharing (disarankan: 1200x630px)

### Hero & Banner
- `hero-bg.jpg` - Background image untuk hero section (disarankan: 1920x1080px)

### Gallery & Content
- `smk-metland-building.jpg` - Foto gedung sekolah (disarankan: 800x600px)
- `news-1.jpg`, `news-2.jpg`, `news-3.jpg` - Gambar untuk berita (disarankan: 800x500px)
- `testimonial-1.jpg` - Foto untuk testimoni (disarankan: 200x200px)

### Partnership Logos (Opsional)
Gambar logo mitra kerjasama dapat ditambahkan sesuai kebutuhan.

## Catatan

- Website akan tetap berfungsi tanpa gambar (menggunakan placeholder)
- Format gambar yang disarankan: JPG untuk foto, PNG untuk logo dengan transparansi
- Optimalkan ukuran file untuk performa website yang lebih baik
- Gunakan gambar dengan kualitas tinggi tetapi ukuran file yang reasonable
